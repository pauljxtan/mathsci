Class BulStoAdaptive

source code

            BulStoAdaptive

Integrates a system of ODEs using the Bulirsch-Stoer method, with adaptive step sizes.

This class is a little different from the others since the integration is performed recursively:

  1. There is no step() method
  2. The initial conditions are passed to the integrate() method instead of the constructor

Resursively performs the steps of adaptive Bulirsch-Stoer.

Parameters:
  • duration (number) - total simulation time
  • delta (number) - desired accuracy per unit time
  • t0 (number) - initial time
  • X0 (vector) - initial system state (default=[0.0, 0.0, ...])
  • n_dims (number) - number of dimensions (required if X0 is not given)
  • nmax (number) - maximum subdivisions of current time step
