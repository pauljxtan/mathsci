binomial_coefficient(n, k)

source code 

Computes the coefficient of the x^k term in the binomial expansion of (1+x)^n. This quantity is also referred to as "n choose k" in combinatorics, that is, the number of ways to select k elements from a set of n elements. The family of binomial coefficients form Pascal's triangle, by way of Pascal's rule.

Parameters:
  • k (number)
  • n (number)
Returns: number
